# 文件批量重命名 (FileRenamer)

一个基于 Android 16 的现代化文件批量重命名工具，采用液态玻璃（Glassmorphism）主题设计。

## ✨ 功能特点

- **📂 浏览文件** — 通过系统文件选择器浏览任意目录（内部存储、U盘、SD卡）
- **✅ 批量选择** — 逐项勾选或一键全选/取消全选
- **🔀 拖拽排序** — 长按文件左侧拖拽手柄，拖动调整文件处理顺序
- **✏️ 13种重命名操作**：
  - **添加前缀** — 在文件名前添加指定文本
  - **添加后缀** — 在文件名后（扩展名前）添加指定文本
  - **删除前N个字符** — 删除文件名开头的N个字符
  - **删除后N个字符** — 删除文件名末尾的N个字符（保留扩展名）
  - **替换前N个字符** — 将文件名开头的N个字符替换为指定文本
  - **替换后N个字符** — 将文件名末尾的N个字符替换为指定文本（保留扩展名）
  - **从前往后第N位插入** — 在文件名指定位置插入文本
  - **从前往后第N位删除** — 从文件名指定位置开始删除N个字符
  - **从后往前第N位插入** — 从文件名末尾往前数指定位置插入文本
  - **从后往前第N位删除** — 从文件名末尾往前数指定位置开始删除N个字符
  - **前缀id号** — 按文件列表顺序自动添加编号前缀（01_、02_...）
  - **后缀id号** — 按文件列表顺序自动添加编号后缀（_01、_02...）
  - **从前往后第N位替换** — 从文件名指定位置开始替换N个字符为新字符串
  - **从后往前第N位替换** — 从文件名末尾往前数指定位置开始替换N个字符
  - **替换空格** — 将文件名中的所有空格替换为指定字符串（留空则删除空格）
- **📊 进度条** — 重命名时显示实时进度（当前处理数/总数）
- **🎨 液态玻璃主题** — 毛玻璃效果、半透明渐变、圆角设计
- **🌓 深浅色主题** — 跟随系统自动切换
- **🖼️ 壁纸取色** — 从桌面壁纸提取主色调，动态调整主题色
- **📱 自适应图标** — 液态玻璃风格的应用图标

## 🚀 构建与安装

### 环境要求

- Android SDK (API 36)
- JDK 17+
- Gradle 9.6.1+

### 构建步骤

```bash
# 1. 设置 Android SDK 路径
export ANDROID_HOME=/path/to/android/sdk
export ANDROID_SDK_ROOT=$ANDROID_HOME

# 2. 构建 Debug APK
./gradlew assembleDebug

# 3. 构建 Release APK（已签名）
./gradlew assembleRelease

# 4. APK 输出位置
app/build/outputs/apk/debug/app-debug.apk
app/build/outputs/apk/release/app-release.apk
```

### Termux 环境构建

在 Android Termux 中构建时，需要安装 `aapt2` 包：

```bash
pkg install aapt2 gradle
```

项目已配置 `android.aapt2FromMavenOverride` 指向 Termux 本地的 `aapt2`。

### 安装 APK

```bash
# 使用 termux-open 安装
termux-open app/build/outputs/apk/release/app-release.apk
```

## 📖 使用说明

### 基本流程

1. **打开应用** → 点击「选择文件夹」按钮
2. **选择目录** → 系统弹出文件选择器，选择要操作的文件夹
3. **勾选文件** → 点击文件/目录前的复选框进行选择
4. **拖拽排序**（可选）→ 长按文件左侧的 **☰ 拖拽手柄**，上下拖动调整文件处理顺序
5. **批量操作** → 点击底部「批量重命名」按钮
6. **选择类型** → 从15种操作中选择一种
7. **输入参数** → 根据操作类型输入相应参数
8. **确认执行** → 点击「开始重命名」，等待进度条完成

### 操作类型详解

#### 1. 添加前缀
在文件名开头添加指定文本，不修改扩展名。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| `笔记_` | `文档.pdf` | `笔记_文档.pdf` |

#### 2. 添加后缀
在文件名末尾（扩展名前）添加指定文本。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| `_v2` | `文档.pdf` | `文档_v2.pdf` |

#### 3. 删除前N个字符
删除文件名开头的N个字符（仅操作主名部分，保留扩展名）。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| `3` | `ABCDE.pdf` | `DE.pdf` |

#### 4. 删除后N个字符
删除文件名末尾的N个字符（仅操作主名部分，保留扩展名）。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| `3` | `ABCDE.pdf` | `AB.pdf` |

#### 5. 替换前N个字符
将文件名开头的N个字符替换为指定文本。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| 3→`XX` | `ABCDE.pdf` | `XXDE.pdf` |

#### 6. 替换后N个字符
将文件名末尾的N个字符替换为指定文本。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| 3→`XX` | `ABCDE.pdf` | `ABXX.pdf` |

#### 7. 从前往后第N位插入
从文件名开头数，在第N个位置插入指定文本。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| 位置2, 插入`XX` | `ABCDE.pdf` | `ABXXCDE.pdf` |

#### 8. 从前往后第N位删除
从文件名开头数，从第N个位置开始删除M个字符。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| 位置1, 删除2个 | `ABCDE.pdf` | `ADE.pdf` |

#### 9. 从后往前第N位插入
从文件名末尾往前数，在第N个位置插入指定文本。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| 位置2, 插入`XX` | `ABCDE.pdf` | `ABCXXDE.pdf` |

#### 10. 从后往前第N位删除
从文件名末尾往前数，从第N个位置开始删除M个字符。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| 位置2, 删除2个 | `ABCDE.pdf` | `ABE.pdf` |

#### 11. 前缀id号
按文件在列表中的排列顺序，自动添加编号前缀。编号位数根据文件数量自动调整（<10个用1位，<100个用2位，以此类推）。

| 文件列表顺序 | 原文件名 | 新文件名 |
|------------|---------|---------|
| 第1个 | `文档.pdf` | `01_文档.pdf` |
| 第2个 | `照片.jpg` | `02_照片.jpg` |
| 第10个 | `笔记.txt` | `10_笔记.txt` |

#### 12. 后缀id号
按文件在列表中的排列顺序，自动添加编号后缀。

| 文件列表顺序 | 原文件名 | 新文件名 |
|------------|---------|---------|
| 第1个 | `文档.pdf` | `文档_01.pdf` |
| 第2个 | `照片.jpg` | `照片_02.jpg` |

#### 13. 从前往后第N位替换
从文件名开头数，从第N个位置开始替换M个字符为新字符串。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| 位置1, 替换2个为`XX` | `ABCDE.pdf` | `AXXDE.pdf` |

#### 14. 从后往前第N位替换
从文件名末尾往前数，从第N个位置开始替换M个字符为新字符串。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| 位置2, 替换2个为`XX` | `ABCDE.pdf` | `ABCXX.pdf` |

#### 15. 替换空格
将文件名中的所有空格替换为指定字符串。如果留空则直接删除所有空格。

| 输入 | 原文件名 | 新文件名 |
|------|---------|---------|
| 替换为`_` | `我的 文件.pdf` | `我的_文件.pdf` |
| 留空 | `我的 文件.pdf` | `我的文件.pdf` |

### 拖拽排序

1. 每个文件左侧有一个 **☰ 拖拽手柄** 图标
2. **长按**手柄图标即可拖拽该文件
3. 上下拖动到目标位置后松手
4. 文件顺序决定 id 编号顺序（前缀id号/后缀id号操作时）

### 重命名进度

点击「开始重命名」后，界面会显示：
- **线性进度条**：直观显示当前进度
- **文字提示**：显示"正在重命名 3/10..."等实时信息
- 完成后自动刷新文件列表并显示结果摘要

## 🎨 主题定制

应用主题会根据系统深浅色模式自动切换，同时从当前桌面壁纸提取主色调：

- **浅色模式** → 浅紫色液态玻璃主题
- **深色模式** → 深紫色液态玻璃主题
- **壁纸取色** → 根据壁纸主色调动态调整主题色
